Tina,

Although the best way is to submit to each directory manually, directories pass so little juice, that the time involved isn’t worth it.

I suggest using directory submission services like <a href="https://www.wlmarketing.com/" rel="nofollow">https://www.wlmarketing.com/</a> – It’s automated, so the quality is down, but you can get several thousand directory submissions quickly and cheap.

Also, you should provide several different descriptions/anchors so that you still retain some variance. Hope that helps! Let me know if you have any more questions.
